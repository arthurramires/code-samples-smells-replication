# Data Directory

## Provenance

All data in this directory were generated by the automated pipeline described in `../INSTALL.md`.

### Collection Date
- Pipeline execution: February 14, 2026
- GitHub API data reflects repository state as of that date

### Data Flow

```
GitHub API (343 repos)
    │
    ├── 01_setup_macos.sh ─── Environment setup
    ├── 02_expand_dataset.sh ── Repository list collection
    ├── 03_run_pipeline.sh ─── Tool execution
    │       │
    │       ├── Designite Java ──→ Raw CSV outputs (code smells + metrics)
    │       │                      318 repos analyzed successfully
    │       │                      25 repos skipped (IC/EC criteria)
    │       │
    │       └── csDetector ──────→ Raw CSV outputs (community metrics)
    │                              50 repos analyzed successfully
    │                              268 repos failed (insufficient GitHub data)
    │                              25 repos skipped (same as Designite)
    │
    └── 04_consolidate.py ──→ Consolidated CSVs (this directory)
```

### Inclusion/Exclusion Filtering
- **343** repositories initially collected
- **25** excluded by IC/EC criteria (LOC < 500 or > 100k, non-Java, archived, forks)
- **318** successfully analyzed by Designite Java
- **50** successfully analyzed by csDetector (subset of the 318)

## Files

| File | Rows | Columns | Description |
|------|------|---------|-------------|
| `consolidated_code_smells.csv` | 318 | 28 | Code Smell counts per repository (24 smell types + totals) |
| `consolidated_metrics.csv` | 318 | 17 | Software quality metrics per repository |
| `consolidated_community.csv` | 50 | 20 | Community metrics and Community Smell indicators |
| `consolidated_full.csv` | 50 | 62 | Inner join of all three files above |

## Format

- **Encoding**: UTF-8
- **Delimiter**: Comma (`,`)
- **Header**: First row contains column names
- **Missing values**: Empty cells (no sentinel values)
- **Numeric format**: Decimal point (`.`), no thousands separator
